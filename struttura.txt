STRUTTURA SITO WEB HUGÒ - Find your happiness

TIPO DI SITO
------------
- Progressive Web App (PWA) mobile-first
- Sito statico HTML/CSS/JavaScript vanilla
- Design responsive con frame desktop che mostra iframe mobile
- Target: Pub/Ristorante con menu digitale

TECNOLOGIE UTILIZZATE
---------------------
- HTML5
- CSS3 (file: style.css)
- JavaScript vanilla (inline nelle pagine)
- Service Worker (service-worker.js) per cache offline
- PWA Manifest (manifest.json)
- Font: Google Fonts - Montserrat (400, 600, 700)
- Icone: Font Awesome 6.5.0 (CDN)
- SEO: robots.txt, sitemap.xml, meta tags Open Graph/Twitter

STRUTTURA PAGINE HTML
---------------------

TUTTE LE PAGINE HANNO:
- Meta tags standard (charset, viewport, description)
- Meta tags PWA (apple-mobile-web-app-*)
- Meta tags Open Graph e Twitter Card
- Link a manifest.json
- Link a style.css
- Link a Google Fonts (Montserrat)
- Link a Font Awesome (CDN)
- Meta tags per disabilitare cache
- Service Worker registration script

PAGINA INDEX.HTML (Shell Desktop)
---------------------------------
- Wrapper desktop con frame mobile (360x780px)
- Iframe che carica home.html
- Service Worker registration
- Design: sfondo nero, frame giallo (#FFC928) con bordo arrotondato

PAGINA HOME.HTML (Homepage)

- Bottom bar fissa con bottoni:

STRUTTURA CSS (style.css)
-------------------------

SEZIONI PRINCIPALI:
1. RESET & BASE
   - Box-sizing, margin, padding reset
   - Scrollbar nascosta
   - Font family: Montserrat
   - Background nero (#000000)
   - Touch action optimization

2. PALETTE COLORI
   - Giallo principale: #FFC928
   - Giallo scuro: #E2AD1F
   - Giallo chiaro: #FFE27A
   - Nero profondo: #111111

3. SHELL DESKTOP (INDEX)
   - .desktop-wrapper: centratura verticale/orizzontale
   - .device-frame: frame mobile (360x780px, border-radius 28px)
   - .device-iframe: iframe full size

4. HOME PAGE (home.html)
   - .page: container principale con gradient radiale
   - .hero: header con logo e testo
   - .cards: sezione cards navigazione
   - .card: card cliccabile con hover effects
   - .contact-block: blocco contatti/orari
   - .site-footer: footer pagina
   - .bottom-bar: barra fissa in basso
   - .scroll-top: bottone scroll to top

5. PAGINE MENU INTERNE
   - .menu-page: container pagina menu
   - .menu-header: header con logo e titolo
   - .menu-close: bottone home fisso
   - .menu-section: sezione menu
   - .menu-item: item singolo prodotto
   - .menu-footer: footer pagina menu

6. RESPONSIVE
   - Media query per tablet (min-width: 768px)
   - Safe area insets per iPhone con notch

SERVICE WORKER (service-worker.js)
----------------------------------
- Strategia: Network First
- Cache name: hugohub-v4 (incrementare versione per aggiornamenti)
- Precaching file principali all'installazione
- Cache runtime per file caricati dinamicamente
- Gestione aggiornamenti automatici
- Supporto offline
